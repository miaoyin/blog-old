自定义函数与内建函数重名，又不想改为，怎么办？加前缀.  

例如：list可以使用__builtin__.list区分
{% highlight python %}
import __builtin__

def list(f):
    '''
    将返回结果转为list
    '''
    def __call(*args, **kwargs):
        return __builtin__.list(f(*args, **kwargs))
    return __call
{% endhighlight %}
